# AI-Smart-Parking-Finder
The Smart AI Parking Finder is an intelligent parking assistance system built using Python and Tkinter, designed to help users find the most efficient parking slot within a parking area.
It simulates a real-world smart parking system by integrating AI pathfinding algorithms (like A* and BFS) with an interactive graph-based visualization of the parking layout.

This project aims to optimize vehicle movement inside parking zones, reduce congestion, and improve user experience through real-time path visualization and decision-making.

How It Works

1.Run the project using python main.py.
2.A visual grid of parking slots appears (green = available, red = occupied).
3.Select your current position and desired parking slot by clicking.
4.The system computes and animates the optimal path.
5.Selecting a new target recalculates and displays the updated route dynamically.
